# Remakeplace Comparing Tools

A web tool for comparing and preparing FFXIV housing market data from reMarket JSON files.

## What This Project Can Do

- Upload and process two reMarket files for side-by-side comparison.
- Normalize furniture and fixture data into a consistent format.
- Show differences between two files, including added, removed, and changed quantities.
- Map dye color values to dye names using local reference data.
- Retrieve item pricing data by selected server.
- Export results in a Teamcraft-compatible format.

## Inspiration

This project was inspired by [FFXIV-Market-Shopper](https://github.com/TrainerHol/FFXIV-Market-Shopper) by TrainerHol.

## Quick Start

```bash
npm install
npm run dev
```
